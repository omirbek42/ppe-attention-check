## 📌 1. Резюме проекта

### 1.1 Описание
Веб-приложение для проверки **Средств Индивидуальной Защиты (СИЗ)** и оценки **внимания/координации** сотрудника с использованием компьютерного зрения. Приложение работает полностью на клиенте (браузер) без отправки данных на сервер, обеспечивая максимальную приватность.

### 1.2 Ключевая идея
**"Никаких серверных возможностей не требуется"** - весь анализ видеопотока происходит локально в браузере через набор моделей машинного обучения (TensorFlow.js, MediaPipe).

### 1.3 Проблема, которую решает
- ❌ **Проблема:** Ручная проверка СИЗ медленная, субъективная, зависит от человека
- ✅ **Решение:** Автоматическая проверка через компьютерное зрение в реальном времени

## 📋 4. Функциональные требования

### 4.1 Основной модуль - Проверка СИЗ

#### 4.1.1 Детекция лица
- **Требование:** Определить наличие лица в кадре
- **Технология:** face-api.js (TinyFaceDetector)
- **Точность:** 98%+
- **Задержка:** ~20ms
- **Показатели:**
  - ✓ Видно / ✗ Не видно
  - Уверенность (%)
  - Координаты лица

#### 4.1.2 Проверка глаз
- **Требование:** Определить открыты ли ОБА глаза
- **Точность:** 85-90%
- **Алгоритм:**
  - Анализ координат век (68 ориентиров лица)
  - Расстояние между веками > 6% от ширины глаза
  - Минимум 4 пикселя между веками
- **Показатели:**
  - ✓ Открыты / ✗ Закрыты
  - ✗ Левый закрыт / ✗ Правый закрыт
  - Индивидуальное определение каждого глаза

#### 4.1.3 Детекция очков
- **Требование:** Определить надеты ли защитные очки
- **Технология:** Анализ пикселей (яркость)
- **Точность:** 70-80%
- **Алгоритм:**
  - Извлечение региона вокруг глаз
  - Подсчёт тёмных пикселей (< 100 яркости)
  - Если > 25% пикселей тёмные → очки надеты
- **Ограничения:**
  - Не работает с прозрачными очками
  - Требует хорошего освещения
- **Показатели:**
  - ✓ Надеты / ✗ Не видны
  - % тёмных пикселей

#### 4.1.4 Детекция каски
- **Требование:** Определить надета ли каска
- **Технология:** Анализ пикселей (верхняя часть головы)
- **Точность:** 65-75%
- **Алгоритм:**
  - Анализ региона над лбом
  - Подсчёт тёмных пикселей
  - Если > 40% пикселей тёмные → каска надета
- **Ограничения:**
  - Не работает с белыми/светлыми касками
  - Зависит от освещения сверху
- **Показатели:**
  - ✓ Надета / ✗ Не видна

#### 4.1.5 Общий статус СИЗ
- **Требование:** Определить готовность работника
- **Логика:**
  - 🟢 ГОТОВ = Лицо видно И оба глаза открыты И очки надеты И каска надета
  - 🟠 ТРЕБУЮТСЯ ИСПРАВЛЕНИЯ = любой из параметров не соответствует
  - 🔵 ПРОВЕРКА = идет анализ
- **Визуализация:**
  - Большой статус-блок с цветом
  - Текст описания требуемых действий
  - Обновляется в реальном времени (30 fps)

### 4.2 Модуль тестирования - Тест внимания

#### 4.2.1 Структура теста
- **Количество заданий:** 5
- **Время на задание:** ~3-5 сек
- **Общее время:** ~15-20 сек
- **Повторяемость:** Неограниченно

#### 4.2.2 Задания

**Задание 1: Показать 2 пальца**
- Инструкция: "Покажите 2 пальца"
- Детекция: MediaPipe Hands
- Проверка: totalFingers == 2
- Успех: ✓ (зелёный)
- Fail: ✗ (красный)

**Задание 2: Показать 3 пальца**
- Инструкция: "Покажите 3 пальца"
- Проверка: totalFingers == 3

**Задание 3: Показать 4 пальца**
- Инструкция: "Покажите 4 пальца"
- Проверка: totalFingers == 4

**Задание 4: Закрыть левый глаз**
- Инструкция: "Закройте левый глаз"
- Проверка: leftEyeOpen == false AND rightEyeOpen == true
- Успех: ✓ Левый глаз закрыт

**Задание 5: Закрыть правый глаз**
- Инструкция: "Закройте правый глаз"
- Проверка: rightEyeOpen == false AND leftEyeOpen == true
- Успех: ✓ Правый глаз закрыт

#### 4.2.3 Система оценки
```
X/5 заданий выполнено = (X/5) * 100%

100% → 🏆 Идеально
80-99% → 🎉 Отлично
60-79% → 👍 Хорошо
40-59% → ⚠️ Нужна практика
<40% → 💪 Потренируйтесь
```

#### 4.2.4 Отчет результатов
- Список выполненных/невыполненных заданий
- Общий результат (X/5)
- Процент успеха (%)
- Текстовая интерпретация
- Возможность повторить тест

## 🏗️ 5. Нефункциональные требования

### 5.1 Безопасность
- ✅ **100% локальная обработка** - видеопоток не отправляется на сервер
- ✅ **Без отслеживания** - нет аналитики, нет сбора данных
- ✅ **HTTPS не требуется** - но приветствуется
- ✅ **GDPR совместимо** - нет передачи персональных данных
- ✅ **Работает с локальным сервером** - никакие данные не передаются

### 5.2 Приватность
- Видеопоток анализируется только в памяти браузера
- Результаты хранятся только локально (IndexedDB)
- Пользователь может очистить всё нажатием на "Очистить кэш"
- Нет никаких логов на сервере

### 5.3 Надежность
- Обработка ошибок для всех сценариев
- Graceful degradation если модели не загружены
- Резервная обработка при недостатке памяти
- Логирование ошибок в консоль браузера

### 5.4 Масштабируемость
- На одном компьютере: 1-10 браузеров одновременно
- На разных компьютерах: неограниченно
- Каждый браузер независим (0 серверных ресурсов)

### 5.5 Совместимость
- ✅ Windows 7+ (с современным браузером)
- ✅ Mac OS X 10.13+
- ✅ Linux (все дистрибутивы)
- ✅ Android 8+ (в Chrome)
- ✅ iOS 12+ (в Safari)

### 5.6 Доступность
- Контраст текста >= 4.5:1 (WCAG AA)
- Все интерактивные элементы имеют фокус стили
- Поддержка клавиатурной навигации (планируется)
- Alt текст для всех изображений

### 6.4 Модели машинного обучения

#### face-api.js
- **TinyFaceDetector**: Быстрая детекция лиц (на базе MobileNet)
  - Входные данные: Canvas 640x480
  - Выходные данные: Координаты лиц + уверенность
  - Скорость: ~20ms per frame
  
- **FaceLandmark68Net**: 68 ориентиров лица
  - Входные данные: Detection box
  - Выходные данные: 68 точек (глаза, нос, рот и т.д.)
  - Скорость: ~20ms
  
- **FaceExpressionNet**: Выражение лица
  - 7 эмоций (neutral, happy, sad и т.д.)
  - Для будущих расширений

#### MediaPipe Hands
- **HandLandmarker**: 21 точка ориентира на руке
  - Входные данные: Video frame
  - Выходные данные: Координаты пальцев
  - Скорость: ~15ms per frame
  - Поддерживает: Обе руки одновременно

### 6.5 Алгоритмы анализа

#### Определение открытости глаз
```javascript
// Координаты век (face-api точки 36-47)
leftEyeDistance = |eyeTop.y - eyeBottom.y|
threshold = eyeWidth * 0.06 + 4

isOpen = (leftEyeDistance > threshold)
```

#### Подсчёт поднятых пальцев
```javascript
// MediaPipe Hand Landmarker
fingerTips = [4, 8, 12, 16, 20]
fingerPIPs = [3, 7, 11, 15, 19]

for each finger:
  if (landmarks[tip].y < landmarks[PIP].y):
    count++
```

#### Детекция очков
```javascript
// Область вокруг глаз
darkPixels = count(brightness < 100)
darkRatio = darkPixels / totalPixels

glassesDetected = (darkRatio > 0.25)
```

#### Детекция каски
```javascript
// Область над лбом
darkPixels = count(brightness < 120)
darkRatio = darkPixels / totalPixels

helmetDetected = (darkRatio > 0.4)
```
