# 🛡️ Система проверки СИЗ и внимания

Веб-приложение для проверки средств индивидуальной защиты (очки, каска, открытость глаз) и теста на внимание/координацию с использованием компьютерного зрения. Вся обработка видео происходит локально в браузере (face-api.js + MediaPipe Hands) — видеопоток никогда не покидает устройство.

Реализовано по [PRD.md](PRD.md).

## Запуск локально

Откройте `index.html` через любой локальный HTTP-сервер (нужен для загрузки ML-моделей и доступа к камере), например:

```
python -m http.server 8080
```

затем перейдите на `http://localhost:8080/`.

## Технологии

- face-api.js (детекция лица, 68 ориентиров)
- MediaPipe Hands (детекция рук и пальцев)
- Chart.js (график истории проверок)
- Vanilla JS / HTML / CSS — без сборки и серверной части
